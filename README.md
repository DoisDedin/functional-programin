# functional-programin

Comandos basicos para se fazer a criação / manipulação do projeto na linguem HASKELL 

//criar um novo projeto
stack new [nome]

//entrar na pasta  apos a criação do projeto
cd [nome]

//organizar o projeto
stack setup

//abrir visual studio code
code .

//compilar o projeto
stack build

//executar o projeto
stack exec [name]-exe
